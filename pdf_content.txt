1. System Overview
Project Summary
The Fitness & Wellness Management System (FWS) is a custom Frappe application built on top of ERPNext v15+. It covers the
complete fitness business value chain — from member onboarding and subscription management through class scheduling, trainer
management, equipment maintenance, health assessments, nutrition planning, and billing — integrating natively with ERPNext's
Accounts, HR, Stock, and CRM modules.

5 phases / ~22 weeks
Core Modules at a Glance
 Module
 Scope
 Member & Subscription
Member registry, subscription plans, renewals, attendance, freeze/cancel
Class & Schedule
Class types, timetables, batch enrollment, trainer assignment, waitlists
Trainer & Staff
Trainer profiles, certifications, availability, performance, commission
Facility & Equipment
Equipment register, maintenance schedules, AMC contracts, locker mgmt
Health & Fitness Assessment
Body metrics, fitness tests, goal tracking, progress reports
Nutrition & Diet
Diet plans, meal templates, supplement tracking, nutrition goals
Billing & Finance
Plan invoices, POS walk-in payments, EMI, refunds, daily cash reconciliation
Reports & Analytics
Revenue, retention, attendance, trainer performance, P&L; dashboards

Proration & EMI Calculation Engine
Core billing logic lives in fitness_wellness/utils/calculations.py. Called on Member Subscription before_save.
# fitness_wellness/utils/calculations.py
import frappe
from frappe.utils import flt, date_diff, add_months, today
def calculate_prorated_amount(plan_price, start_date, billing_day=1):
    """
    Calculate prorated amount when subscription starts mid-cycle.
    Returns full amount if starting on or before billing day.
    """
    from datetime import date
    start = frappe.utils.getdate(start_date)
    days_in_month = 30
    if start.day <= billing_day:
        return flt(plan_price)
    days_remaining = days_in_month - start.day + billing_day
    return flt(plan_price * days_remaining / days_in_month)
def generate_emi_schedule(subscription_name, total_amount, no_of_instalments, start_date):
    """
    Generate EMI instalment rows for a given subscription.
    Called via hooks on Member Subscription submit when payment_mode == 'EMI'.
    """
    instalment_amount = flt(total_amount / no_of_instalments, 2)
    schedule = []
    for i in range(no_of_instalments):
        due_date = add_months(start_date, i)
        schedule.append({
            "instalment_no": i + 1,
            "due_date": due_date,
            "amount": instalment_amount,

def calculate_trainer_commission(trainer, period_from, period_to):
    """
    Sum commission-eligible sessions for a trainer in the period.
    """
    trainer_doc = frappe.get_doc("Trainer Profile", trainer)
    # Count group sessions
    group_sessions = frappe.db.count("Class Attendance",
        filters={"trainer": trainer, "session_date": ["between", [period_from, period_to]]})
    # Count PT sessions
    pt_sessions = frappe.db.count("Trainer Assignment",
        filters={"trainer": trainer, "session_date": ["between", [period_from, period_to]]})
    if trainer_doc.commission_type == "Per Session":
        gross = (group_sessions + pt_sessions) * flt(trainer_doc.commission_rate)
    else:
        gross = flt(trainer_doc.commission_rate)
    tds = flt(gross * 0.10)  # 10% TDS
    return {"gross_commission": gross, "tds_deducted": tds,
            "net_payable": gross - tds,
            "group_sessions_taken": group_sessions,
